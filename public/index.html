Personal Project

A Simon Game


Stack

HTML, CSS, JavaScript, jQuery

A "Simon Game" was one of the projects in the course I took. I decided to not follow the directions and simply complete the assignment knowing the basic function of a simon game. I incorporated pop-culture sounds and some distractions to make it difficult. The player can double-click on various background elements and make them slowly change color. Another single-click on that same element will stop the color change.

This was my first solo project. I used jQuery for DOM manipulation, JS for control flow, and css for animations and transitions. The site is built using CSS Grid to ease in scaling to increased number of squares, or to aide in mobile development. Good luck, don't take Arnold too seriously.
