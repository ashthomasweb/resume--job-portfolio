Client Project

Tech

This project was built for a Colorado-based local business. JavaScript Program

Music Trainer & Generator
on-going development


Tech

CSS, ES6, HTML

This project is the conclusion to a hypothesis I had while a music teacher. At that time I did not know anything about programming other than it must follow standard logic principles. In short, I theorized that if you put all the 'rules' of music theory into code and allowed for randomization in it's choices, the resultant music would be surprisingly familiar and even sound pretty good at times- with only occasional awkward moments. This project is two-fold, containing a set of practice tools, as well as a music generator. The tools help a student train their ear in the subjects of melody, relative pitch, pitch memory, and tempo.

Work your klangfarbenmelodie chops with multiple orchestral instrument choices.




Full-stack Template

LumberJack


Tech

Node.js, Express, EJS, Prismic, Curator, NodeMailer

"LumberJack" is a full-stack template developed for small businesses and social media creators. The theme is lightweight and integrates third-party services to eliminate update, security, and code-heavy issues. Featuring a headless CMS with Prismic, a multi-network aggregator in Curator, and automatic email responses by NodeMailer, LumberJack's needs for maintenance and updates are low. Ships with detailed setup instructions accesible by any front-end developer or diligent person.

Making use of third-party API's keeps the website owner from having to engage with the site code after launch, keeping breakage to a minimum. LumberJack's blog is automatically generated using EJS and comes with a "Custom Type" ready to upload to your Prismic repository. Curator allows owners and creators to build feeds from more than one network, enabling more cross-network audience engagement.

Designed for a two-color brand scheme with highly legible greyscale copy. Card elements are modular and can be used in a variety of configurations, color and size. LumberJack is fully compatible with any device.

Personal Project

The Arnold Game


Tech

HTML, CSS, JavaScript, jQuery

"The Arnold Game" was one of the projects in the course I took. I decided to not follow the directions and simply complete the assignment knowing the basic function of a simon game. I incorporated pop-culture sounds and some distractions to make it more fun and difficult. The player can double-click on various background elements and make them slowly change color. Another single-click on that same element will stop the color change.

This was my first solo project. I used jQuery for DOM manipulation, JS for control flow, and CSS for animations and transitions. The site is built using CSS Grid to ease in scaling to increased number of squares, or to aid in mobile development. Good luck, don't take Arnold too seriously.
