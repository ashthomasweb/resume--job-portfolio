Productivity Application

Flow Post
heroku-hosted


Tech

Node.js, React, Firebase for Auth and Storage

Flow Post is a skewmorphic organization application. Users can save custom boards and layouts. Board and note colors can be set by user and individual notes can be checked off. A special 'Mat' function creates a box that always remains behind the notes, serving as a visual container. Future versions will include a 'Share' function, multi-select, mat-board grouping, and full type customization.

Built with react. Atypical design paradigms were necessary for the skewmorphic effect during the 'drag' operation. Resultant architecture is difficult to maintain and extend. Refactorization using Hooks and Context may be possible and will be attempted before futher development, but full display update during the drag operation must be present to fulfill project requirements.

JavaScript Program

Music Trainer & Generator
on-going development


Tech

ES6, WebAudio, Local Storage

This project is the conclusion to a hypothesis I had while a music teacher. At that time I did not know anything about programming other than it must follow standard logic principles. In short, I theorized that if you put all the 'rules' of music theory into code and allowed for randomization in it's choices, the resultant music would be surprisingly familiar and even sound pretty good at times- with only occasional awkward moments. JavaScript and the WebAudio API are used to provide a web-based ear-trainer and infinite music generator.

This project is two-fold, containing a set of practice tools, as well as a music generator. The tools help a student train their ear in the subjects of melody, relative pitch, pitch memory, and tempo. The music generator is a passion project that I will continue to build upon for some time. My more recent study in object-oriented principles leads me to realize the entirety of the project needs redesign and overhaul to move very far forward. However, the underlying logic in how to sort through the musical information has been solved. In time it will not only be an expression of the power of western musical structures, but also a learning device.

The application has a high level of educational value for a student or professional. Certain skills that were difficult to practice have been made available and accentuated through the use of the trainers. Work your klangfarbenmelodie chops with multiple orchestral instrument choices. The music trainer includes a function which allows for different instruments to randomly substitute the primary instrument sound, while still asking the student to repeat the pattern back. This develops the ear to be sensitive to pitch while listening to instruments outside of their familiarity. The melodic idea consisting of the same note, but different tones/instruments, is called "klangfarbenmelodie". This melody trainer facilitates the study of an invaluable skill that is difficult to methodically practice.





Full-stack Template

LumberJack Theme
heroku-hosted


Tech

Node.js, Express, EJS, Prismic, Curator, NodeMailer

"LumberJack" is a full-stack template developed for small businesses and social media creators. The theme is lightweight and integrates third-party services to eliminate update, security, and code-heavy issues. Featuring a headless CMS with Prismic, a multi-network aggregator in Curator, and automatic email responses by NodeMailer, LumberJack's needs for maintenance and updates are low. Ships with detailed setup instructions accessible by any front-end developer or diligent person.

Making use of third-party API's keeps the website owner from having to engage with the site code after launch, keeping breakage to a minimum. LumberJack's blog is automatically generated using EJS and comes with a "Custom Type" ready to upload to your Prismic repository. Curator allows owners and creators to build feeds from more than one network, enabling more cross-network audience engagement.

Designed for a two-color brand scheme with highly legible greyscale copy. Card elements are modular and can be used in a variety of configurations, color and size. LumberJack is fully compatible with any device.
